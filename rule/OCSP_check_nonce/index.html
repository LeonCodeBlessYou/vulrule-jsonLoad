Description: OCSP_check_nonce() returns the result of the nonce comparison between B<req> and B<resp>. The return value indicates the result of the comparison. If nonces are present and equal 1 is returned. If the nonces are absent 2 is returned. If a nonce is present in the response only 3 is returned. If nonces are present and unequal 0 is returned. If the nonce is present in the request only then -1 is returned.

Label: return value check
