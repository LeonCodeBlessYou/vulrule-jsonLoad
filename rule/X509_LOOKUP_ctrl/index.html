AURC - openssl - X509_LOOKUP_ctrl

Return to Homepage

Rule Profile

Description: X509_LOOKUP_ctrl() returns -1 if the B<X509_LOOKUP> doesn't have an associated B<X509_LOOKUP_METHOD>, or 1 if the X<509_LOOKUP_METHOD> doesn't have a control function. Otherwise, it returns what the control function in the B<X509_LOOKUP_METHOD> returns, which is usually 1 on success and 0 in error.

Label: return value check

Parameter Index: N/A

CWE Type：CWE-253

QLCode:

import cpp

class OpenSSLFunctionCall extends FunctionCall {
  OpenSSLFunctionCall() {
    this.getTarget().hasName("X509_LOOKUP_ctrl")
  }
}

from OpenSSLFunctionCall call, UnaryOperation uop
where
  uop.getOperator() = "!" and
  uop.getOperand() = call.getAnAccess()
